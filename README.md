# Selamat Hari Raya Aidilfitri 🕌✨

A beautiful, interactive Hari Raya greeting website with animated fireworks and festive sound effects.

## Features

🎆 **Animated Fireworks** - Continuous firework display with colorful explosions
🎵 **Sound Effects** - Festive celebration sounds and firework effects
✨ **Interactive** - Click anywhere to create your own fireworks
🌙 **Festive Design** - Traditional green and gold color scheme
📱 **Responsive** - Works beautifully on all devices

## Usage

Simply open `index.html` in your browser or visit the live site.

Click the sound button (🔊/🔇) in the top-right corner to toggle sound effects.
